# AirBnB Clone - The Console

## Project Description
This is the first step toward building a full-stack web application: an AirBnB clone. In this phase, we are developing a command-line interface (CLI) to manage the back-end objects of the application. 

The goal is to handle the initialization, serialization, and deserialization of objects using a JSON file storage system.

## The Command Interpreter
The console is a tool that allows us to manage application objects. You can:
* Create new objects (e.g., User, Place, City).
* Retrieve objects from the JSON storage file.
* Perform operations (count, show, etc.).
* Update attributes of existing objects.
* Delete objects.
